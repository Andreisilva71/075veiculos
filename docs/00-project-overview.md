# Visão Geral do Projeto

## Nome do Projeto
CarLead - Landing Page de Compra, Venda e Troca de Veículos

## Objetivo
Desenvolver uma landing page moderna para captura de leads de uma loja automotiva, permitindo que usuários:

- Comprem veículos
- Vendam veículos
- Solicitem troca de veículos

O sistema deve coletar informações através de um formulário interativo (multi-step form) e enviar os dados para um backend em Node.js.

---

# Stack Tecnológica

## Frontend
- HTML5
- CSS3
- JavaScript
- Vue.js

## Backend
- Node.js
- Express.js

---

# Funcionalidades Principais

## Landing Page
- Hero Section
- Sessão de benefícios
- Sessão de depoimentos
- Formulário interativo

## Formulário Multi-Step
O usuário responderá perguntas passo a passo sobre:
- Interesse (compra/venda/troca)
- Tipo de veículo
- Faixa de preço
- Tempo para fechar negócio

## Captura de Leads
Ao final do fluxo:
- Nome
- WhatsApp
- E-mail

Os dados serão enviados para o backend utilizando requisições HTTP.
